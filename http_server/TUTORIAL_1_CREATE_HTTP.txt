=================================== СОЗДАНИЕ ПРОЕКТА ===============================================

1. Создаем папку проекта

2. Заходим в эту папку, в консоли вводим npm init. В диалоге можем выбрать параметры проекта, такие как имя и т.п. Вся эта информация попадет в созданный файл package.json

В этом же файле см. пункт "main". В нем указывается входной файл для нашего приложения, обычно index.js

3. Создаем index.js в той же папке.

4. В этом файле создаем http-сервер.

const http = require('http')

const hostname = 'localhost'  
(localhost = имя хоста для локального компьютера)

const port = 3000
(с помощью номера порта мы обозначаем конкретную серверную программу в локальной сети в качестве запущенного процесса)

const server = http.createServer((req, res) => {
  res.end('Welcome to Node!')
})
(в этой константе будет храниться объект сервера, который мы вызываем через метод createServer(). этот метод уже зашит в модуль http)

В качестве аргумента этот метод принимает коллбэк-функцию, которая позволяет нам обозначить параметры сервера.

С помощью метода .end(), который мы вызываем на объекте res, мы обозначаем, что ответ от сервера получен. В качестве аргумента мы указываем сообщение, которое будет содержаться в ответе.

server.listen(port, hostname, () => {
  console.log(`Server running at ${hostname}:${port}`)
})

В этом методе последним аргументом мы вызываем коллбэк, который будет вызываться, когда сервер будет запущен.

ВСЕ, НАШ СЕРВЕР ГОТОВ

5. Запускаем сервер из консоли. Чтобы это сделать, мы должны сказать ноде, какой файл запустить.

node index

СЕРВЕР ЗАПУЩЕН, консоль выдаст Server running at localhost:3000

6. Создаем в package.json скрипт для запуска сервера

  "scripts": {
    "start": "node index"
  },
