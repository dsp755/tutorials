1. Чтобы добавить условие в разметку JSX (напр. если 1, показывать <Text>, если 0, показывать <View>), можно 
использовать следующую структуру:

{todos.length ? <Flatlist> ... </Flatlist> : <Text> ... </Text>}

если todos.length не равно 0, будет показан элемент FlatList и все его дочерние элементы, а если равно 0, то Text.
