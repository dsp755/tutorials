1. Пропсы по умполчанию.

Если мы создаем свой компонент, то в возможные для него пропсы мы можем передать дефолтное значение:

export const AppButton = ({ children, color = THEME.MAIN_COLOR, onPress }) => ( ... )


При этом если мы импортируем и вставим данный компонент в другой компонент, мы можем переназначить это значение:

<AppButton color={THEME.RED_COLOR} onPress={() => removeTodo(todo.id)}>Remove</AppButton>


Если мы не передадим этот пропс в такой импортируемый компонент, его значение будет равно дефолтному значению.
