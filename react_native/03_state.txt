1. Импортируем { useState } в компонент:

import React, { useState } from 'react';

State создается по следующей схеме:

const [todos, setTodos] = useState([])

Теперь todos получает значени [].
Чтобы поменять это значение, надо вызвать функцию setTodos(НОВОЕ ЗНАЧЕНИЕ)

При этом значение todos изменится на НОВОЕ ЗНАЧЕНИЕ.


2. Частичное изменение стейта:

  const addTodo = (title) => {
    setTodos(prev => [ ...prev, { id: Date.now().toString(), title }])
  }

Метод setTodos принимает аргументом последнее состояние стейта (в нашем случае - массива todos),
а далее мы преобразовывем это последнее состояние в новый массив, развернув в нем ...prev и добавив в него новый объект.

Далее мы передаем эту функцию addTodo в качестве пропса в тот компонент, в котором хотим ее использовать.

<AddTodo onSubmit={addTodo} />
