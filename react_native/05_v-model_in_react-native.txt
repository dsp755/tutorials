1. У некоторых компонентов, напр. TextInput, есть аттрибут value, который привязывает его значение к значению какой-то переменной.
Например, если у нас в state есть переменнаю something, мы можем привязать значение компонента TextInput к этой переменной:
value={something}

В остальных случаях запись значения в переменную по вводу производится через state.

Например, для компонента <TextInput> есть аттрибут onChangeText={text => someFunction(text)}

Уже в этой функции someFunction происходит запись введенного значения в state, а оттуда мы можем это значение 
использовать как перемунную.
