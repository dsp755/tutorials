1. в корневой папке приложения создаем папку db, а в ней файл index.js

2. в файле

const { Pool } = require('pg');

const pool = new Pool({
  user: 'node_user',
  host: 'localhost',
  database: 'monstersdb',
  password: '1',
  port: 5432
});

- так мы создаем функцию, которая будет служить основным способом взаимодействия с базой данных
например, она включает метод query

3. теперь в том же файле мы можем создать первый запрос к нашей базе данных

pool.query('SELECT * FROM monsters', (err ,res) => {
  if (err) return console.log(err);

  console.log(res);
});


4. чтобы запустить этот файл, можно из корневой папки проекта запустить следующую команду

node db

- эта команда автоматически в папке db найдет файл index.js и выполнит в нем команды

В консоли будет выведен объект с запрашиваемыми из дб данными.
